Microsoft Windows Re-Install.

Microsoft Windows Re-Installation requires all the data/files and software on your computer or laptop to be deleted. We highly advise that you perform your own backup before we attempt to re-install Microsoft Windows, as we cannot be held liable for any data, file, or software loss.

On customer request, we will be able to attempt a backup of your files (pictures, documents, music and videos) and then restore these after Microsoft Windows re-installation. In order to back up your files, we will strongly advise using a backup device provided by the customer (such as external hard disk drive, Flash drive etc.). We will also need to be informed by the client of the location of files that need to be backed up. Please be aware that software that has been installed on Microsoft Windows cannot be backed up, and will require a reinstallation using original software installation media. Customer must have the valid product /registration key of Microsoft Windows and softwares need to be installed.

During the Microsoft Windows re-installation there is always a chance that hardwares such as Ram or HDD/SDD or Motherboard or processor may be found faulty. In that case, we cannot be held liable for the same.

As with all our services we highly recommend that you take your data backup prior to handing over for repair or service. Data backup will remain the client’s sole responsibility. During the repair or service, we cannot be held liable for any loss of data file information etc., however we will try out very best to limit the risk of any data loss while performing a service or repair.
